# Hermes Skills by SilentSpiral

个人 Hermes Agent 技能合集。

## 可用技能

### Email
- **himalaya-outlook-oauth2** — Outlook/Office365 邮箱通过 OAuth2 连接 Himalaya CLI

### Productivity
- **everything-wsl2** — 从 WSL2 调用 Windows Everything 搜索引擎

## 安装

```bash
hermes skills tap add SilentSpiral/hermes-skills
hermes skills install himalaya-outlook-oauth2
```
